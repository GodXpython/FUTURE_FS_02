## 📌 Task Overview
Built a complete Client Lead Management System (Mini CRM) to track and manage leads generated from website contact forms, with full CRUD operations, status tracking, and note-taking functionality.

---

## ✨ Features Implemented
- ✅ **Dashboard Stats** – Total, New, Contacted, Converted lead counts
- ✅ **Lead Listing** – Name, email, phone, source, status, date added
- ✅ **Add Lead** – Modal form with validation
- ✅ **Edit Lead** – Update any lead's information
- ✅ **Delete Lead** – Remove leads with confirmation
- ✅ **Status Updates** – Dropdown to switch between New / Contacted / Converted
- ✅ **Follow-up Notes** – Slide-in panel to add and view notes per lead
- ✅ **Search & Filter** – Search by name/email, filter by status tab
- ✅ **CSV Export** – Download all leads as a CSV file
- ✅ **Local Storage** – Data persists across browser refreshes
- ✅ **Toast Notifications** – Instant feedback for all actions
- ✅ **Responsive Sidebar Layout** – Professional admin dashboard design
- ✅ **Sample Data** – Preloaded with 5 sample leads on first load
